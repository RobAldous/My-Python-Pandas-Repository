## Python Data Analysis Project

This project was completed during a Data Technician bootcamp and focuses on using Python and pandas to explore, clean, and analyse retail and sales data.

### Key Skills Demonstrated

- **Data Filtering & Subsetting (pandas)**  
  Applied filtering techniques to isolate relevant data and extract insights. Used `.loc[]` and `.iloc[]` for precise row and column selection, enabling efficient exploration of large datasets.

- **Data Cleaning & Preparation**  
  Ensured data quality by handling missing values using methods such as `dropna()` and `fillna()`. Performed structured data transformations to prepare datasets for analysis.

- **Data Manipulation (pandas methods)**  
  Utilised powerful pandas functions including:
  - `groupby()` for aggregating sales data and identifying trends  
  - `sort_values()` to rank and organise key metrics  
  - Additional transformations to reshape and structure data for deeper analysis  

- **Data Visualisation**  
  Created visual representations of data using built-in pandas plotting and libraries such as **matplotlib**. This helped communicate patterns, trends, and key findings clearly.

### Project Application

Working with retail and sales datasets, I used Python to:
- Explore data and identify key performance indicators  
- Clean and transform raw data into a usable format  
- Analyse sales trends and customer behaviour  
- Generate visual insights to support business decision-making  

### Outcome

Through this project, I developed the ability to:
- Work efficiently with real-world datasets  
- Apply data analysis techniques using Python and pandas  
- Transform raw data into meaningful insights  
- Present findings through clear visualisations
